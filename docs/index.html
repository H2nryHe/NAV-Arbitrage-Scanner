NAV Arbitrage Scanner (NAVScan)

Daily CEF NAV premium/discount relative-value scanner with staged pipeline, risk notes, and T+1/T+3/T+5 reversion tracking.

Quickstart

python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -r requirements.txt

export PATH="$PWD/scripts:$PATH"
navscan run --date 2026-02-20 \
  --config configs/default.yaml \
  --universe configs/universe_stage6_smoke.yaml \
  --output-dir reports_stage6_smoke

See examples under examples/outputs/ (frozen demo outputs).

Outputs

reports/date=YYYY-MM-DD/top_opportunities.csv
reports/date=YYYY-MM-DD/daily_report.md

Why it’s “desk-style”

  • Staged pipeline (fetch → standardize → candidates → report → reversion tracking)
  • Risk flags + “insufficient history / event data partial” surfaced in report
  • Designed for daily run & review (Action artifacts + summary)
